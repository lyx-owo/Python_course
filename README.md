1.作業簡介:

本專案旨在透過 Git 與 GitHub 版本控制系統，紀錄與管理本組成員的 Python 課堂練習歷程。

我們將 Git 版本管控運用於「個人程式碼的集中管理」與「團隊非同步開發」上。透過建立共享的 GitHub Repository，我們將每週的 Python 課堂練習上傳至各自的資料夾或對應的進度分支中。這不僅讓我們能安全地備份並追蹤每次程式碼的修改歷程，也為組員間互相觀摩寫法與交流技術提供了一個協作平台。

.md檔案**：說明文件與學習筆記
